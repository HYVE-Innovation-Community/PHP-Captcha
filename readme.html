Die Captcha Class

Gonna captch'em all!

This class lets you easily create some Captchas for securing your logins or register or any else formulars you want protect against robots or spam.

How to use?

  1. Insert an image-tag at the website which has a php-file as source.
  2. Initialise the CaptchaClass in the php-file which u use as image.
  3. Call the generateImage-function in the instance.
  4. The php-file which validates the formData needs to use the CaptchaClass and use the compareCaptcha-function, to validate if the entered captcha equals the generated picture.

Requirements:

  • SESSION have to be started before initialising the Class
  • You need a Font File for rendering the Text.

How to set the Font-File as using Font (4 options):

  • Use the first parameter of the constructor:
    $captcha = new Captcha('FreeSans.ttf');
    $captcha->generateImage();
  • Use the second parameter of the generateImage-Function:
    $captcha = new Captcha();
    $captcha->generateImage('FreeSans.ttf');
  • Set it with the setFont-Function:
    $captcha = new Captcha();
    $captcha->setFont('FreeSans.ttf');
    $captcha->generateImage();
  • Set it manually:
    $captcha = new Captcha();
    $captcha->font = 'FreeSans.ttf';
    $captcha->generateImage();
